# Driverforge Homebrew tap

Homebrew tap for the [Driverforge](https://driverforge.dev) CLI — build,
deploy, and manage Control4 drivers.

## Install

```
brew tap driverforge/tap
brew install driverforge
```

Homebrew (6.0 and later) will refuse to install from a third-party tap until
you explicitly trust it — this is standard for all non-official taps. When
brew asks, trust the tap and re-run the install:

```
brew trust driverforge/tap
```

## What you're trusting

This repository is intentionally minimal, so it's easy to review:

- It contains only the `driverforge` cask, generated by
  [GoReleaser](https://goreleaser.com) and committed automatically by our
  release pipeline on each release. No human pushes to this repository.
- The cask downloads binaries from `releases.driverforge.com` (our release
  CDN), verified by the SHA-256 checksums pinned in the cask — nothing is
  built or fetched from GitHub at install time.

## Upgrading

```
brew upgrade driverforge
```
