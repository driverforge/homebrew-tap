# Driverforge Homebrew tap

Homebrew tap for [Driverforge](https://driverforge.dev) command-line tools:

- **driverforge** — build, deploy, and manage Control4 drivers
- **gayle** — deploy configuration and secrets to AWS SSM Parameter Store and
  Azure Key Vault

## Install

```
brew tap driverforge/tap
brew install driverforge
brew install gayle
```

Homebrew (6.0 and later) will refuse to install from a third-party tap until
you explicitly trust it — this is standard for all non-official taps. When
brew asks, trust the tap and re-run the install:

```
brew trust driverforge/tap
```

## What you're trusting

This repository is intentionally minimal, so it's easy to review:

- It contains only the casks for the tools listed above, generated by
  [GoReleaser](https://goreleaser.com) and committed automatically by each
  tool's release pipeline. No human pushes to this repository.
- The casks download binaries from `releases.driverforge.com` (our release
  CDN), verified by the SHA-256 checksums pinned in each cask — nothing is
  built or fetched from GitHub at install time.

## Upgrading

```
brew upgrade driverforge
brew upgrade gayle
```
